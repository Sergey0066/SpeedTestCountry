# SpeedTestCountry

Утилита для проверки **скорости скачивания** с тестовых серверов в разных странах.

## Запуск

Запуск из корня папки:

```bash
"SpeedTest Sergey0066.exe"
```

## Режимы запуска

- **Меню** (без параметров): выбор региона / `all` / `custom`
- **Один регион**:

```bash
"SpeedTest Sergey0066.exe" --region ru
```

- **Все регионы (`all`)**: прогоняет все страны по очереди (по 15 сек на страну) и показывает итоговую таблицу

```bash
"SpeedTest Sergey0066.exe" --region all
```

- **Свой URL (`custom`)**: тестирует скачивание по вашему URL (файл)

```bash
"SpeedTest Sergey0066.exe" --region custom --url "https://example.com/file.bin"
```

## Параметры

- **`--threads N`**: количество потоков скачивания (по умолчанию `8`)
- **`--region`**: код региона (`ru`, `us`, …) или `all/custom`
- **`--url`**: URL файла (только для `custom`)

## Что показывается во время теста

### Один регион

Во время теста на экране видно:

- **Регион**: выбранная страна
- **Сервер**: какой тестовый сервер выбран
- **URL**: откуда идёт скачивание
- **Потоки**: сколько потоков скачивает одновременно
- **Текущее**: скорость “прямо сейчас” (Mbps и рядом MB/s)
- **Среднее**: средняя скорость с начала теста (Mbps и рядом MB/s)
- **Скачано**: сколько данных скачано (MiB)
- **Время**: прошло / всего секунд

### `all` (все страны)

Пока идёт прогон, сверху отображается таблица со статусом по странам (что уже прошло и средняя скорость), а снизу — текущая страна и текущие показатели.

## Итог после `all`

В конце `all` печатается только таблица:

- **Страна — Среднее / Max (Mbps)**

## Цвета скорости (подсветка чисел)

Числа скорости подсвечиваются так:

- **красный**: до 10 Mbps
- **оранжевый**: до 50 Mbps
- **жёлтый**: до 100 Mbps
- **зелёный**: до 500 Mbps
- **синий**: выше 500 Mbps
